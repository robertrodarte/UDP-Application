# UDP-Application
## How To Run
1. On the server, open a terminal
2. Navigate to the "UDP-Application" folder and run the following command: `python udp_application.py`
3. Enter "1" to run a server
4. On the client, open a terminal
5. Navigate to the "UDP-Application" folder and run the following command: `python udp_application.py`
6. Enter "2" to run a client
7. Repeat steps 4-6 on other client devices
8. Begin sending messages
